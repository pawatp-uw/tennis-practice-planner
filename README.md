# Weather Aware Tennis Practice Planner

This project is a Flask web application that helps users generate a structured
tennis practice session based on their available time, skill level, focus area,
and current weather conditions.

## Features
- Interactive form for user input
- Practice session generator
- Weather-based recommendations using Open-Meteo API
- Equipment reminders

## How to run

## 1. Create virtual environment

python3 -m venv venv  
source venv/bin/activate  

## 2. Install dependencies

pip install -r requirements.txt  

## 3. Run the app

python3 app.py  

## 4. Open in browser

http://127.0.0.1:5000

## API
This project uses Open-Meteo:
